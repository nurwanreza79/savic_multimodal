# Results and Artifact Guide

This folder should contain manuscript-supporting outputs.

## Recommended contents

- `summary_semantic.csv`
- `summary_baselines.csv`
- `compare_semantic_vs_baselines.csv`
- `summary_ablation.csv`
- manuscript tables
- plots
- qualitative figures
- selected demo MP4 outputs

## Minimal artifact set for a journal submission

1. The exact CSV summaries used to create the paper tables.
2. The exact plots shown in the paper.
3. The qualitative visual and audio figures used in the manuscript.
4. One or more shareable demo reconstructions.
5. A short note describing which file corresponds to which table or figure in the paper.
